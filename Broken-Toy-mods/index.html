Ward Call Prioritisation Game

Original work by Dr Simon McCormick.
Released under CC BY-NC-SA license.

Instructions:

1. Review the pending tasks displayed and which ward they have originated from.
2. Decide which task should be completed first.
3. To complete a task, click it.
4. A new task will be issued for you to reevaluated the next task to be completed.
5. Remember, in real life, you don't want to keep tasks waiting too long.
